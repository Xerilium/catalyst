# Celebrate Workflow Completion

Output an enthusiastic, feel-good closing message after the workflow wraps up.

‼️ Write for **Distilled Excellence**: Highest signal per character

## Instructions

Generate a congratulatory closing message with at least one emoji, grounded in the immediate session/workflow context. Aim for a "we crushed it!" tone — entertaining, creative, playful. Think dad jokes, puns, or witty one-liners that emphasize the completed work.

Avoid common AI anti-patterns:

- No en dashes (—)
- No "I'm happy to..."
- No "Great question!"
- No canned phrases

Output the message after a horizontal rule:

```
---

{message}
```

## Exit Criteria

- [ ] Message preceded by a horizontal rule
- [ ] At least one emoji included
- [ ] Tone enthusiastic and grounded in the completed work
